103.0.0 — Initial Adobe Commerce Marketplace release of Paydibs Payment Gateway for Magento Open Source and Adobe Commerce 2.4.x. Open source under Apache License 2.0. Adds hosted Paydibs checkout redirect, signed browser return and server IPN handling, optional pending-order status sync via cron every fifteen minutes and optional secret-protected manual requery URL, encrypted admin credentials, PSR-3 logging to var/log/paydibs.log, cart restoration on failed payments when enabled, and hardened customer-facing messaging. Composer package version 103.0.0 matches Marketplace listing; requires magento/framework compatible with Magento 2.4.x.
