103.0.0 — Initial Adobe Commerce Marketplace release of Paydibs Payment Gateway for Magento Open Source and Adobe Commerce 2.4.x. Adds hosted Paydibs checkout redirect, signed return and IPN handling, optional pending-order query via cron, encrypted admin credentials, PSR-3 logging to var/log/paydibs.log, and cart restoration on failed payments when enabled. Hardens storefront and admin messaging and aligns with Magento 2.4 framework and payment module constraints.
